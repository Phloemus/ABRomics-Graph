# ABRomics KG 

Abromics KG correspond à la partie T2.5 (WP2) du projet ABRomics. Le 
but de cette partie est de rendre les données le plus FAIR possible 
grâce à l'intégration d'un graphe de connaissance au projet ABRomics.

## WARNING 

This repo is currently not pushed remotely. A maintainer or Owner of 
the project Abromics Kg should create the main branch before any commit 
can reach the remote repository
